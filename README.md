# CommissioningEngine_V1_Final

Vite + React + Tailwind application for thermodynamic commissioning data collection.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Included features

- Local persistence via `localStorage`
- JSON export
- CSV export
- Responsive industrial dark mode UI
- Thermodynamic engine and failure matrix integration
